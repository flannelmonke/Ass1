JavaScript

JavaScript is a scripting language used for web applications embedded in HTML pages.

Data Types Control Constructs Functions Arrays

Data Types

JavaScript has several built-in data types, including String, Number, Bigint, Boolean, Undefined, Null, Symbol and Object. These data types are used to store and manipulate different types of values in JavaScript.

Key Takeaways

  • String is used to hold arrays of characters for sentences or words.
  • Number is used to hold numeric values.
  • Bigint is used to hold large integers bigger than the default value of the previously mentioned Number data type.
  • Boolean is used to hold true or false values.
  • Undefined is used to represent an undefined value.
  • Null is used to represent a null value.
  • Symbol is used to create unique identifiers for objects.
  • Object is used to hold key-value pairs of data.

Control Constructs

Fundamental components to programming. Control constructs in JavaScript include loops, conditionals, and functions. These constructs allow you to control the flow of your program, execute code conditionally, and repeat code multiple times. As well as defining reusable blocks of code for abstracting problems you want to solve one time.

Key Takeaways

  • if; else; else if; statements are used to execute code conditionally. (True or False)
  • loops(for; while; do while; for of;) are used to repeat code multiple times, the number of times to repeat these blocks are flexible and defined by you, the programmer.
  • Functions are used to define reusable blocks of code that can be called anywhere in your program. They are used to abstract problems you want to solve one time. You can use arrow functions or function declarations to define functions.

Functions

Functions are reusable blocks of code that can be called anywhere in your program. They allow you to abstract problems you want to solve one time and reuse the solution multiple times. Functions can take parameters and return values, making them flexible and powerful tools for organizing and structuring your code.

Key Takeaways

  • Abstraction: Functions allow you to abstract problems you want to solve once.
  • Parameter: Functions can take parameters, allowing you to pass data into the function to customize its behavior.
  • Return values: Functions can return values, allowing you to get data back from the function after it has executed.

Arrays

Arrays are used to store multiple values in a single variable. They are a powerful data structure that allows you to group related data together and manipulate it as a single unit. Arrays can contain any type of data, including strings, numbers, objects, and even other arrays.

Key Takeaways

  • Data Storage: Arrays in JavaScript are used for storing multiple values in a single variable, making them versatile data structures.
  • Dynamic Size: Unlike some other programming languages, JavaScript arrays can dynamically adjust their size, allowing elements to be added or removed as needed.
  • Indexed Access: Elements in an array are accessed using numeric indices, with the first element typically having an index of 0.
  • Iteration: Arrays support iteration through elements using loops like for, forEach, or map, enabling efficient processing of array elements.
