Angular Component Example

In Angular, a component is a fundamental building block that controls a part of the user interface (UI). Each component consists of three main parts:

  1. Template: Defines the HTML view of the component.
  2. Class: Contains the logic of the component, including properties and methods.
  3. Metadata: Provides Angular with information about the component, such as its selector and template.
Here's a basic example: 


                import { Component, NgModule } from '@angular/core';
                import { BrowserModule } from '@angular/platform-browser';
                
                // Define a component with a selector and a template
                @Component({
                    selector: 'app-root',  // HTML tag to use this component
                    template: '<h1>Hello, Angular Component!</h1>'  // HTML content to be displayed
                })
                export class AppComponent { }  // The component class with logic (empty in this example)
                
                // Define an Angular module to declare and bootstrap the component
                @NgModule({
                    declarations: [AppComponent],  // Declare the component in the module
                    imports: [BrowserModule],      // Import BrowserModule to run Angular in a browser
                    bootstrap: [AppComponent]       // Bootstrap the component to start the application
                })
                export class AppModule { }  // The module class

Explanation:

@Component Decorator:

  • selector: Specifies the custom HTML tag that will be used to embed this component.
  • template: Defines the HTML content that this component will display. In the code, we use <h1> and </h1> to escape HTML tags and show them as text.

AppComponent Class:

  • Contains the logic for the component (empty here, but typically includes properties and methods).

@NgModule Decorator:

  • declarations: Lists all components that belong to this module.
  • imports: Imports other Angular modules needed by this module.
  • bootstrap: Specifies the root component that Angular should bootstrap when starting the application.

Angular Routing Example

Angular routing allows you to navigate between different views or pages in a single-page application (SPA). It uses routes to define paths and associate them with components.

Here's a simple example of setting up routing: 


                import { NgModule } from '@angular/core';
                import { RouterModule, Routes } from '@angular/router';
                import { BrowserModule } from '@angular/platform-browser';
                
                // Define routes for different paths
                const routes: Routes = [
                    { path: 'home', component: HomeComponent },  // Route for home component
                    { path: 'about', component: AboutComponent },  // Route for about component
                ];
                
                // Define an Angular module for routing
                @NgModule({
                    imports: [RouterModule.forRoot(routes), BrowserModule],  // Import RouterModule and configure routes
                    exports: [RouterModule]  // Export RouterModule for use in other parts of the app
                })
                export class AppRoutingModule { }  // The routing module class

Explanation:

  • RouterModule: Provides routing capabilities.
  • Routes: Defines an array of route configurations.

Routes Array:

  • Each route maps a URL path (e.g., 'home', 'about') to a component (e.g., HomeComponent, AboutComponent).

@NgModule Decorator for Routing Module:

  • imports: Configures routes using RouterModule.forRoot(routes) to set up the routing in the root module.
  • exports: Makes RouterModule available to other parts of the application.

In addition to the routing configuration, use the following HTML in your templates for navigation:


            Home | About

routerLink creates links to different routes, while router-outlet acts as a placeholder where routed components will be displayed.
