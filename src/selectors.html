Tag Selectors

This section demonstrates the use of tag selectors in CSS.

This is a paragraph.

This is a div.
This is a span.

In this example, we select all <div> elements using the document.querySelectorAll('div') method. Then, we use the forEach method to iterate over each selected element and change its text color to blue using the style.color property.

ID Selectors

This section demonstrates the use of ID selectors in CSS.

This is a unique paragraph.

In this example, we select an element with the ID "uniqueParagraph" using the document.getElementById method. Then, we set the font-weight CSS property of the selected element to bold, making the text bold.

Class Selectors

This section demonstrates the use of class selectors in CSS.

This paragraph has red text.

This paragraph has blue text.

In this example, we select elements with the class "redText" and "blueText" using the document.querySelectorAll('.className') method. Then, we use the forEach method to iterate over each selected element and apply corresponding styles.

Attribute Selectors

This section demonstrates the use of attribute selectors in CSS.

In this example, we select input elements with the type attribute "text" using the document.querySelectorAll('[attributeName]') method. Then, we apply styles to these selected elements.

Attribute Value Selectors

This section demonstrates the use of attribute value selectors in CSS.

In this example, we select button elements with the type attribute value "submit" using the document.querySelectorAll('[attributeName="value"]') method. Then, we apply styles to these selected elements.

Attribute Selectors


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Tag Selectors</title>
    <style>
        div {
            color: blue; /* CSS selector for div */
        }
    </style>
</head>
<body>
    <p>This is a paragraph.</p>
    <div>This is a div.</div>
    <span>This is a span.</span>

    <script>
        document.querySelectorAll('div').forEach(element => {
            element.style.color = 'blue';
        });
    </script>
</body>
</html>

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ID Selectors</title>
    <style>
        #uniqueParagraph {
            font-weight: bold; /* CSS selector for unique paragraph */
        }
    </style>
</head>
<body>
    <p id="uniqueParagraph">This is a unique paragraph.</p>

    <script>
        const uniqueParagraph = document.getElementById('uniqueParagraph');
        uniqueParagraph.style.fontWeight = 'bold';
    </script>
</body>
</html>

ID Selectors


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Class Selectors</title>
    <style>
        .redText {
            color: red; /* CSS selector for red text */
        }
        .blueText {
            color: blue; /* CSS selector for blue text */
        }
    </style>
</head>
<body>
    <p class="redText">This paragraph has red text.</p>
    <p class="blueText">This paragraph has blue text.</p>

    <script>
        document.querySelectorAll('.redText').forEach(element => {
            element.style.color = 'red';
        });

        document.querySelectorAll('.blueText').forEach(element => {
            element.style.color = 'blue';
        });
    </script>
</body>
</html>

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Attribute Selectors</title>
    <style>
        input[type="text"] {
            background-color: lightgray; /* CSS selector for text input */
        }
    </style>
</head>
<body>
    <input type="text" value="This is a text input">

    <script>
        document.querySelectorAll('input[type="text"]').forEach(element => {
            element.style.backgroundColor = 'lightgray';
        });
    </script>
</body>
</html>
